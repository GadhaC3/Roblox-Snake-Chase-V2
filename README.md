# Project Overview
This project is a Roblox game called Snake Chase. Players control a snake that grows longer as it eats food, and must avoid running into walls or themselves.

# Features
- **Multiplayer Support**: Play with friends or random players.
- **Customizable Snake**: Players can choose different skins and colors for their snake.
- **Dynamic Environments**: Unique maps that change every game.
- **Leaderboards**: Keep track of top scores.

# Quick Start Guide
1. Download the project files from the repository.
2. Open Roblox Studio.
3. Import the downloaded files into your workspace.
4. Run the game and start playing!

# Official Roblox API Implementation
This project uses the following APIs from Roblox:
- `DataStoreService`: For saving player stats.
- `RemoteEvent`: For communication between server and clients.
- `TweenService`: To animate objects smoothly in the game.

Make sure to check the [Roblox Developer Hub](https://developer.roblox.com/en-us/api-reference) for more details on the APIs used.
